[size=6][b]Bannerlord.GameMaster (BLGM)[/b][/size][size=4][b][url=https://github.com/SolWayward/Bannerlord.GameMaster/wiki]
Complete Documentation on GitHub Wiki
[/url][/b][/size]
Console commands and power tools mod for Mount & Blade II: Bannerlord providing game management through the game console.

[b]Tested With:[/b] Bannerlord 1.3.9, 1.3.10, 1.3.12 beta, and works with or without Warsails.

[b]BLGM[/b] extends Bannerlord's console with powerful commands for managing heroes, clans, kingdoms, items, and game state. This mod is useful for taking control of your game, testing things out, fixing saves, or whatever other reason you may need to take control of your game. 

The purpose of this mod is mainly for my own use as a foundation for another mod I am working on it but it can also be useful to modders for testing, debugging, or to quickly implement functionality in their mods as well, an API is also provided. The mod will regular be updated with additional features and more powerful functionality.

All commands use the `gm.` prefix for easy organization.  
All commands can be used by targeting entity names or entity Ids. A query system is also included for quickly searching for entities.

[size=5][b]Key Features[/b][/size]
[list]
[*][b]Hero Management[/b] - Modify attributes, gold, health, relationships, life state
[*][b]Clan Management[/b] - Control membership, finances, renown, leadership
[*][b]Kingdom Management[/b] - Handle diplomacy, settlements, clan membership
[*][b]Item Management[/b] - Full inventory/equipment control with quality modifiers
[*][b]Equipment Save/Load[/b] - Save and load hero equipment sets to files
[*][b]Advanced Queries[/b] - Powerful search with AND/OR logic, sorting, filtering
[*][b]Command Logging[/b] - Track all command usage for debugging
[*][b]Testing Framework[/b] - Automated validation system
[/list]
[size=5][b]Installation[/b][/size]
1. Download and install manually or via mod manager
2. Enable in Bannerlord launcher
3. Press [b]~[/b] or [b]`[/b] in-game to open console

[b]No external dependencies required[/b] - This mod uses only native Bannerlord APIs

[size=5][b]Quick Start[/b][/size]
[b]Command Structure:[/b]
[font=Courier New]gm.<category>.<command> [parameters][/font]
[b]
Example Commands:[/b]
[code]gm.hero.set_gold player 50000
gm.query.hero empire lord female
gm.query.item sword tier5 sort:value:desc
gm.item.add imperial_sword 5 player
gm.item.save_equipment_both player my_loadout
gm.clan.set_gold clan_empire_1 100000
gm.kingdom.declare_war empire battania
gm.log.enable[/code]
[size=5][b]Query System[/b][/size]
Use powerful queries to search and filter:
[list]
[*][b]Basic:[/b] [font=Courier New]gm.query.hero <terms>[/font], [font=Courier New]gm.query.item <terms>[/font]
[*][b]AND logic:[/b] [font=Courier New]gm.query.hero empire lord[/font] (empire AND lord)
[*][b]OR logic:[/b] [font=Courier New]gm.query.hero OR empire battania[/font] (empire OR battania)
[*][b]Sorting:[/b] [font=Courier New]gm.query.item bow sort:value:desc[/font] (descending value)
[*][b]Filtering:[/b] [font=Courier New]gm.query.item armor tier5[/font] (tier 5 armor only)
[/list]
[b]Sort Options:[/b] name, value, tier, type (add :asc or :desc)
