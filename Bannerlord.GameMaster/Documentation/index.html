Getting Started

Note: All commands must be entered in the Bannerlord console. Press ~ or ` to open the console in-game. All GameMaster commands begin with the gm. prefix.

Quick Start Guide

Basic Command Structure
gm.<category>.<command> [parameters]

Replace <category> with hero, clan, kingdom, query, log, or test, and <command> with the specific command name.

Common Usage Examples

Setting Hero Gold
gm.hero.set_gold lord_1_1 10000

Sets the gold of hero with ID "lord_1_1" to 10,000.

Searching for Heroes
gm.query.hero empire lord female

Finds all female lords in the Empire faction.

Declaring War
gm.kingdom.declare_war empire battania

Declares war between the Empire and Battania kingdoms.

Command Conventions

<parameter> Required
Parameters enclosed in angle brackets are required for the command to execute.
[parameter] Optional
Parameters enclosed in square brackets are optional and can be omitted.
hero_id String
Hero identifiers can be partial names or StringIds. The system will search for matches.
clan_id String
Clan identifiers can be partial names or StringIds. The system will search for matches.
kingdom_id String
Kingdom identifiers can be partial names or StringIds. The system will search for matches.

Important Notes

Warning: Many commands make permanent changes to your game state. It is recommended to:
  • Create save backups before using commands
  • Test commands in a separate save file first
  • Enable logging with gm.log.enable to track all command usage
  • Some commands cannot be undone (e.g., killing heroes, destroying clans)
