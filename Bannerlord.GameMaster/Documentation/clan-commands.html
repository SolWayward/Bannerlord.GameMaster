Clan Management Commands

Commands for managing clans in Bannerlord. These commands allow you to add heroes to clans, manage gold distribution, set renown and tier levels, change clan leadership, and destroy clans.

Note: All clan commands require you to be in campaign mode. Many operations permanently modify clan state, so use with caution and consider backing up your save first.

Hero Management

gm.clan.add_hero HERO
Adds a hero to a specified clan, transferring them from their current clan (if any).

Parameters

clan string Required
Clan identifier (StringId or partial name match)
hero string Required
Hero identifier to add (StringId or partial name match)

Examples

Add a hero to a clan
gm.clan.add_hero empire_south lord_1_1
Transfer wanderer to clan
gm.clan.add_hero battania wanderer_empire_1

Returns

Success: "{Hero Name} transferred from '{Previous Clan}' to '{Clan Name}'."

Gold Management

gm.clan.add_gold GOLD
Adds gold to a clan by distributing it evenly among all living clan members.

Parameters

clan string Required
Clan identifier (StringId or partial name match)
amount integer Required
Total gold amount to distribute among clan members

Examples

Add 50,000 gold to a clan
gm.clan.add_gold empire_south 50000

Returns

Success: "Added {amount} gold to {Clan Name} (distributed among {member_count} members)."
"Clan gold changed from {previous_gold} to {new_gold}."
Note: Gold is distributed evenly among all living clan members. Any remainder is distributed one gold at a time to members until exhausted.
gm.clan.set_gold GOLD
Sets the clan's total gold by first zeroing all member gold, then distributing the specified amount evenly.

Parameters

clan string Required
Clan identifier (StringId or partial name match)
amount integer Required
Target total clan gold (minimum: 0)

Examples

Set clan total gold to 100,000
gm.clan.set_gold empire_south 100000
Zero out clan gold
gm.clan.set_gold rebel_clan 0

Returns

Success: "Set {Clan Name}'s gold to {amount} (distributed among {member_count} members)."
"Previous clan gold: {previous_gold}, New clan gold: {new_gold}."
gm.clan.add_gold_leader GOLD
Adds gold directly to the clan leader only, without distributing to other members.

Parameters

clan string Required
Clan identifier (StringId or partial name match)
amount integer Required
Gold amount to add to clan leader

Examples

Add 50,000 gold to clan leader
gm.clan.add_gold_leader empire_south 50000

Returns

Success: "Added {amount} gold to {Leader Name} (leader of {Clan Name})."

"Clan total gold: {previous_clan_gold} â†’ {new_clan_gold}"

Gives gold to a specific clan member rather than distributing to all members.

Parameters

Clan identifier (StringId or partial name match)
hero string Required
Clan member identifier (StringId or partial name match)
amount integer Required
Gold amount to give to the hero

Examples

Give gold to specific clan member

Prestige & Renown Management

gm.clan.set_renown PRESTIGE
Sets the clan's renown to a specific value. Renown affects clan tier and reputation.

Parameters

clan string Required
Clan identifier (StringId or partial name match)
amount float Required
New renown value (minimum: 0)

Examples

Set clan renown to 500
gm.clan.set_renown empire_south 500

Returns

Success: "{Clan Name}'s renown changed from {old_renown} to {new_renown}."
gm.clan.add_renown PRESTIGE
Adds renown to a clan's current amount. Use negative values to subtract renown.

Parameters

clan string Required
Clan identifier (StringId or partial name match)
amount float Required
Amount to add (positive) or subtract (negative)

Examples

Add 100 renown
gm.clan.add_renown empire_south 100
Subtract 50 renown
gm.clan.add_renown rebel_clan -50

Returns

Success: "{Clan Name}'s renown changed from {old_renown} to {new_renown} (+{amount})."
gm.clan.set_tier PRESTIGE
Sets the clan's tier level by adjusting renown to the appropriate threshold. Tier affects party size limits and companion capacity.

Parameters

clan string Required
Clan identifier (StringId or partial name match)
tier integer Required
Target tier (range: 0-6)

Examples

Set clan to tier 5
gm.clan.set_tier empire_south 5

Returns

Success: "{Clan Name}'s tier changed from {old_tier} to {new_tier}."
Important: Due to Bannerlord's clan tier system, tiers can only be increased, not decreased. Attempting to lower a tier will result in an error. This is a game engine limitation.

Clan Structure Management

gm.clan.set_leader STRUCTURE
Changes the clan's leader to a different member of the clan.

Parameters

clan string Required
Clan identifier (StringId or partial name match)
hero string Required
New leader identifier (must be a clan member)

Examples

Change clan leader
gm.clan.set_leader empire_south lord_1_1

Returns

Success: "{Clan Name}'s leader changed from {Previous Leader} to {New Leader}."
Note: The new leader must already be a member of the specified clan. Returns an error if the hero is not in the clan.
gm.clan.destroy STRUCTURE
Permanently destroys and eliminates a clan from the game. All clan properties and fiefs are lost.

Parameters

clan string Required
Clan identifier to destroy (StringId or partial name match)

Examples

Destroy a clan
gm.clan.destroy rebel_clan_1

Returns

Success: "{Clan Name} has been destroyed/eliminated."
Warning: This action is permanent and cannot be undone. The clan will be marked as eliminated, all fiefs will be transferred, and clan members will lose their affiliation. The player's clan cannot be destroyed.

Quick Reference

Hero Management:
gm.clan.add_hero - Add hero to clan
Gold Management:
gm.clan.add_gold - Add gold distributed to all members
gm.clan.set_gold - Set total clan gold
gm.clan.add_gold_leader - Add gold to leader only
gm.clan.give_gold - Give gold to specific member
Prestige & Renown:
gm.clan.set_renown - Set clan renown
gm.clan.add_renown - Add/subtract renown
gm.clan.set_tier - Set clan tier (0-6, increase only)
Clan Structure:
gm.clan.set_leader - Change clan leader
gm.clan.destroy - Destroy/eliminate clan

Important Notes

Gold Distribution: When adding gold to a clan, it is distributed evenly among all living members. The Clan.Gold property represents the sum of all member gold. Use targeted commands like gm.clan.give_gold to give to specific members.
Clan Tiers: Tier requirements are based on renown thresholds defined by the game's ClanTierModel. Higher tiers provide increased party size, companion capacity, and workshop limits.
