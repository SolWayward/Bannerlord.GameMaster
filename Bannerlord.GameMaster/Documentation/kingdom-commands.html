Kingdom Management Commands

Commands for managing kingdoms in Bannerlord. These commands allow you to add/remove clans from kingdoms, manage diplomatic relations (war and peace), transfer settlements, change rulers, and destroy kingdoms.

Note: All kingdom commands require you to be in campaign mode. Many operations have significant diplomatic and military consequences, so use with caution and back up your save first.

Clan Management

gm.kingdom.add_clan CLAN
Adds a clan to a kingdom, making them a vassal. If the clan is already part of another kingdom, they will leave that kingdom first.

Parameters

clan string Required
Clan identifier (StringId or partial name match)
kingdom string Required
Target kingdom identifier (StringId or partial name match)

Examples

Add a clan to a kingdom
gm.kingdom.add_clan clan_battania_1 empire
Transfer clan between kingdoms
gm.kingdom.add_clan aserai_clan vlandia

Returns

Success: "{Clan Name} joined {Kingdom Name} from {Previous Kingdom}."
Note: If the clan is already part of the target kingdom, the command will return an error. A clan automatically leaves their previous kingdom when joining a new one.
gm.kingdom.remove_clan CLAN
Removes a clan from their current kingdom, making them an independent faction.

Parameters

clan string Required
Clan identifier to remove (StringId or partial name match)

Examples

Remove a clan from their kingdom
gm.kingdom.remove_clan clan_empire_south_1

Returns

Success: "{Clan Name} removed from {Kingdom Name}."
Warning: Cannot remove the ruling clan from a kingdom. The ruling clan must remain part of the kingdom. To change the ruling clan, use gm.kingdom.set_ruler first.

Diplomatic Relations

gm.kingdom.declare_war DIPLOMACY
Declares war between two kingdoms, setting them as hostile factions. This affects all clans and heroes within both kingdoms.

Parameters

kingdom1 string Required
First kingdom identifier (StringId or partial name match)
kingdom2 string Required
Second kingdom identifier (StringId or partial name match)

Examples

Declare war between two kingdoms
gm.kingdom.declare_war empire battania
Start conflict using full identifiers
gm.kingdom.declare_war vlandia aserai

Returns

Success: "War declared between {Kingdom1 Name} and {Kingdom2 Name}."
Note: A kingdom cannot declare war on itself. If the kingdoms are already at war, the command will return an error. War affects trade, diplomacy, and enables military actions.
gm.kingdom.make_peace DIPLOMACY
Establishes peace between two kingdoms that are currently at war, ending hostilities.

Parameters

kingdom1 string Required
First kingdom identifier (StringId or partial name match)
kingdom2 string Required
Second kingdom identifier (StringId or partial name match)

Examples

Make peace between warring kingdoms
gm.kingdom.make_peace empire battania
End war using partial names
gm.kingdom.make_peace vland aser

Returns

Success: "Peace established between {Kingdom1 Name} and {Kingdom2 Name}."
Note: Kingdoms must currently be at war for this command to work. If they are not at war, the command will return an error. Peace enables trade and diplomatic interactions.

Settlement Management

gm.kingdom.give_fief SETTLEMENT
Transfers a settlement (town or castle) to another kingdom. The settlement will be assigned to a random eligible clan within the target kingdom.

Parameters

settlement string Required
Settlement identifier (StringId, e.g., "town_empire_1")
kingdom string Required
Target kingdom identifier (StringId or partial name match)

Examples

Transfer a town to another kingdom
gm.kingdom.give_fief town_empire_1 battania
Transfer a castle
gm.kingdom.give_fief castle_aserai_2 vlandia

Returns

Success: "{Settlement Name} transferred from {Previous Kingdom} ({Previous Owner Clan}) to {New Kingdom} ({New Owner Clan})."
Note: Only towns and castles can be transferred. Villages are tied to their parent settlement. The settlement will be assigned to a random eligible clan in the target kingdom.
gm.kingdom.give_fief_to_clan SETTLEMENT
Transfers a settlement directly to a specific clan, giving them ownership of the fief.

Parameters

settlement string Required
Settlement identifier (StringId, e.g., "town_empire_1")
clan string Required
Target clan identifier (StringId or partial name match)

Examples

Give a town to a specific clan
gm.kingdom.give_fief_to_clan town_empire_1 clan_battania_1
Transfer castle to player clan
gm.kingdom.give_fief_to_clan castle_aserai_2 player_clan

Returns

Success: "{Settlement Name} transferred from {Previous Kingdom} ({Previous Owner}) to {Clan Name}."
Note: The target clan must have a living leader to receive the settlement. The settlement's kingdom alignment will match the clan's kingdom.

Kingdom Properties

gm.kingdom.set_ruler STRUCTURE
Changes the ruler of a kingdom by setting a new ruling clan. The specified hero becomes both the clan leader (if not already) and the kingdom ruler.

Parameters

kingdom string Required
Kingdom identifier (StringId or partial name match)
hero string Required
New ruler identifier (must be part of the kingdom)

Examples

Change kingdom ruler
gm.kingdom.set_ruler empire lord_1_1
Make player the ruler
gm.kingdom.set_ruler vlandia player

Returns

Success: "{Kingdom Name}'s ruler changed from {Previous Ruler} to {New Ruler}."
"Ruling clan is now {Clan Name}."
Note: The hero must be part of the target kingdom (their clan must be a member). If the hero is not currently their clan's leader, they will be promoted to clan leader as well.
gm.kingdom.destroy STRUCTURE
Permanently destroys and eliminates a kingdom from the game. All settlements are transferred and member clans become independent.

Parameters

kingdom string Required
Kingdom identifier to destroy (StringId or partial name match)

Examples

Destroy a kingdom
gm.kingdom.destroy battania

Returns

Success: "{Kingdom Name} has been destroyed/eliminated."
Warning: This action is permanent and cannot be undone. The kingdom will be marked as eliminated, all settlements will be transferred to other kingdoms, member clans become independent, and all diplomatic relations are ended. The player's kingdom cannot be destroyed.

Quick Reference

Clan Management:
gm.kingdom.add_clan - Add clan to kingdom
gm.kingdom.remove_clan - Remove clan from kingdom
Diplomacy:
gm.kingdom.declare_war - Declare war between kingdoms
gm.kingdom.make_peace - Make peace between kingdoms
Settlement Management:
gm.kingdom.give_fief - Transfer settlement to kingdom
gm.kingdom.give_fief_to_clan - Transfer settlement to specific clan
Kingdom Structure:
gm.kingdom.set_ruler - Change kingdom ruler
gm.kingdom.destroy - Destroy/eliminate kingdom

Important Notes

Diplomatic Consequences: Declaring war or making peace has far-reaching effects on trade, party behavior, and AI decision-making. Kingdoms at war will actively raid each other's territories and besiege settlements.
Settlement Transfers: When settlements change ownership, garrison troops, bound villages, and notables remain with the settlement. The new owner clan gains control of all settlement income and management decisions.
Ruling Clan Protection: The ruling clan of a kingdom cannot be removed from that kingdom. To remove them, you must first change the ruler to another clan, then remove the former ruling clan.
